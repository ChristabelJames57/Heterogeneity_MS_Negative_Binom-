# Heterogeneity_MS_Negative_Binom-
This repository contains the code used in the manuscript "Estimating the role of heterogeneities in the first-wave dynamics of COVID-19 in Nigeria", by Christabel James, Jason Matthiopoulos and Daniel T. Haydon. The study develops a method to estimate heterogeneity using Bayesian Markov Chain Monte Carlo for Negative Binomial Observation error.
